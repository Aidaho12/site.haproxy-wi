The "Auto start" is means:

The "Auto start" service provides ability to start the HAProxy service if it has gone out.

For example:

The HAProxy service has been killed for some reasons, e.g. it has done OOM Killer, then the "Auto start" service will detect it and will try to start the HAProxy service.

If you want to use and if you have install HAProxy service manually read how you should set HAProxy service.

The "Auto start" Log

The "Auto start" has it own log in an "Internal logs". You can read more about the "Internal logs" here
