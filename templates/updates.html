RPM


If you use RPM package then just type: 
yum update haproxy-wi
If servers uses proxy to connection Internet add proxy settings in yum.conf: 
vi /etc/yum.conf
proxy=proto://ip:port
Or use web interface for update

How to install RPM read in Installation man

Manual

cd /var/www/haproxy-wi
